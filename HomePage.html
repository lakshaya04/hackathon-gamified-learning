    Data Structures and Algorithms(DSA)is a fundamental part of Computer Science that teaches you how to think and solve complex problems systematically. Using the right data structure and algorithms makes your program run faster,especially when working with lots of data.

    DATA BASE MANAGEMENT SYSTEMS

    Database Management Systems(DBMS) are software systems used to store,retrieve,and run queries on data. A DBMS serves as an interface between and end-user and a database,allowing users to create,read,uodate,and delete data in the data base.

    ARTIFICIAL INTELLIGENCE

    Artificial Intelligence(AI) is a field of science concerned with building computers and machines that can reason,learn,and act in such a way that would normally require human intelligence or that involes data whose scale exceeds what humans can analyze.
